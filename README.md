# Blockchain-balances

# Introduction
User can get to know balances from polygon, ethereum, binance, and avalanche by entering wallet address.

# Run the React:

1. run 'yarn install'.
2. run 'yarn run start'.
3. go to http://localhost:3000'.

# Run the Build:

1. run 'npm install -g serve'.
2. run 'serve -s build'
3. go to 'http://localhost:3000'.
